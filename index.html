// Pipeline

How It Works

01
📸
Image Input
User uploads a plant leaf image via the Tkinter UI or web interface.
02
🔬
Preprocessing
Image resized to 224×224, normalised and fed into the pipeline.
03
🧠
Dual CNN Inference
Model 1 identifies the plant species. Model 2 detects disease class with confidence score.
04
📊
Result Display
Results shown only if confidence ≥ 60% — otherwise "uncertain" is returned.
// Capabilities

Key Features

🌱
Plant Species Identification
MobileNetV2 Model 1 classifies the uploaded leaf into known plant species with high accuracy.
→ MobileNetV2 · Transfer Learning
🦠
Disease Detection
Model 2 specialises in identifying 38+ disease classes across tomato, potato, corn and more.
→ PlantVillage Dataset
🎯
Confidence Thresholding
Predictions below 60% confidence are flagged as uncertain — preventing misleading outputs.
→ Reliable Inference
🖥️
Desktop App (Tkinter)
Fully functional standalone desktop application built in Python Tkinter — no browser needed.
→ Python · Tkinter · PIL
⚡
Fast Inference
MobileNetV2's lightweight depthwise separable convolutions enable rapid predictions on CPU.
→ MobileNetV2 · Keras
📦
Modular Codebase
Clean separation of model loading, preprocessing, UI and inference logic for easy extension.
→ GitHub · Open Source
